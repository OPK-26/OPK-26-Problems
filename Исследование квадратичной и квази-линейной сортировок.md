# Задача: исследование квадратичной и квази-линейной сортировки

## Цель
Экспериментально исследовать и сравнить производительность квадратичного и квази‑линейного алгоритмов сортировки на массивах целых чисел разного размера.

## Задачи

1. Реализовать два алгоритма сортировки:
    * **квадратичный** (на выбор: сортировка выбором, сортировка вставками или сортировка пузырьком);
    * **квази‑линейный** (на выбор: быстрая сортировка, сортировка слиянием или пирамидальная сортировка).

2. Написать код для экспериментального измерения времени работы каждого алгоритма на массивах разной длины.
3. Проверить корректность работы реализованных алгоритмов (написать тесты).
4. Провести серию экспериментов, зафиксировать результаты.
5. Проанализировать зависимость времени работы алгоритмов от размера входного массива.

## Требования к реализации

### Алгоритмы сортировки

**Квадратичный алгоритм** (один из):
* сортировка выбором (`selection_sort`);
* сортировка вставками (`insertion_sort`);
* сортировка пузырьком (`bubble_sort`).

**Квази‑линейный алгоритм** (один из):
* быстрая сортировка (`quick_sort`);
* сортировка слиянием (`merge_sort`);
* пирамидальная сортировка (`heap_sort`).

### Измерение времени

* Использовать `time.perf_counter()` для точного измерения времени выполнения.
* Замерять время **только** выполнения алгоритма сортировки (без учёта генерации массива и проверки корректности).
* Для каждого размера массива проводить **несколько замеров** (рекомендуется 3–5) и брать среднее значение для уменьшения погрешности.

### Проверка корректности

* После каждого запуска алгоритма проверять, что массив действительно отсортирован по возрастанию.
* При обнаружении ошибки — выводить сообщение с указанием размера массива и типа алгоритма.

### Параметры экспериментов

* **Размеры массивов**: выбрать 5–7 значений, равномерно распределённых в диапазоне от 100 до 50 000 (например: 100, 500, 1 000, 2 500, 5 000, 10 000, 50 000).
* **Генерация массивов**: для каждого размера генерировать **случайный массив** целых чисел (например, в диапазоне от 0 до 1 000 000).
* **Количество повторений**: для каждого размера массива провести 3–5 экспериментов и усреднить результаты.
