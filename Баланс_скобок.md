# Задача: проверка баланса круглых скобок в выражении

**Условие**  
Реализовать функцию `is_balanced(expression)`, которая проверяет, сбалансированы ли круглые скобки в заданной строке.

**Требования к функции**  
- Вход: строка `expression` (может быть пустой).  
- Выход: `True`, если скобки сбалансированы; `False` — в противном случае.  
- Учитывать **только круглые скобки** `(` и `)`; остальные символы игнорируются.  
- Пустая строка считается корректной (возвращает `True`).

**Алгоритм работы**  
1. Инициализировать счётчик уровня вложенности скобок (`balance = 0`).  
2. Пройти по каждому символу строки:  
   - если символ — `(`: увеличить `balance` на 1;  
   - если символ — `)`: уменьшить `balance` на 1;  
   - если в какой‑то момент `balance < 0` — вернуть `False` (закрывающая скобка без пары).  
3. После обработки всех символов:  
   - если `balance == 0` — вернуть `True`;  
   - иначе — вернуть `False` (незакрытые открывающие скобки).

**Примеры корректных выражений** (`True`):  
- `""` (пустая строка);  
- `"()"`;  
- `"((()))()(())"`;  
- `"(()())"`.

**Примеры некорректных выражений** (`False`):  
- `")("` (закрывающая скобка перед открывающей);  
- `"((())))"` (лишние закрывающие скобки);  
- `"(()"` (незакрытая открывающая скобка).
