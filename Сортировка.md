# Сортировка (Любая)

## Цель
Реализовать один или несколько алгоритмов сортировки и протестировать их корректность с помощью `assert`.

## Требования к реализации

1. Реализуйте **один из следующих алгоритмов сортировки** (на выбор преподавателя):
   * пузырьковая сортировка (`Bubble Sort`);
   * сортировка выбором (`Selection Sort`);
   * сортировка вставками (`Insertion Sort`);
   * быстрая сортировка (`Quick Sort`);
   * сортировка слиянием (`Merge Sort`);
   * пирамидальная сортировка ('Heap Sort').

2. Функция сортировки должна:
   * принимать на вход сортируемый список и функцию сравнения ключей
   * Изменять принятый массив (не генерировать новый).
   * корректно обрабатывать:
     * пустые списки;
     * списки с одним элементом;
     * списки с повторяющимися элементами;
     * списки, уже отсортированные по возрастанию/убыванию.

3. Напишите **набор тестов** с использованием `assert`, проверяющих:
   * корректность сортировки для обычных случаев (случайные числа);
   * работу с крайними случаями (пустой список, один элемент, два элемента);
   * обработку дубликатов (в том числе, массив состоящий целиком из одного значения, например 20 нулей); 
   * работу с массивами большой длины.
