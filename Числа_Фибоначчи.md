# Задача: Вычисление чисел Фибоначчи

## Условие

Напишите программу, которая вычисляет числа Фибоначчи согласно следующему рекуррентному соотношению:

$$
F(0) = 0, \\
F(1) = 1, \\
F(n) = F(n-1) + F(n-2) \quad \text{для } n > 1.
$$

## Требования к реализации

Необходимо предоставить **два варианта решения**:

1. **Рекурсивный подход** — функция, которая напрямую реализует рекуррентную формулу.
2. **Итеративный подход** — более эффективный алгоритм с использованием цикла и хранения только двух предыдущих значений.

## Входные данные

- Целое неотрицательное число $n$ ($0 \leq n \leq 50$), задающее порядковый номер числа Фибоначчи, которое нужно вычислить.

## Выходные данные

Для каждого из двух подходов выведите:

- Результат вычисления $F(n)$.
- Время выполнения алгоритма (в миллисекундах) — для сравнения эффективности подходов.

## Дополнительно
1. Оцените сложность каждого из подходов.
2. Проверьте свою оценку, построив график времени вычисления от $n$
